About

Controlled Environment Agriculture (CEA) is a data-driven scientific discipline. However, public data on CEA is scarce. Moreover, more and more scientific journals and grants require data dissemination. The CEAOD initiative, started in late 2018 by a diverse group of researchers in CEA, aims to promote data sharing to help accelerate CEA research. There are 3 pillars in this initiative.

  • Data format recomendations: this helps researchers collect data following the best practices and then share data in a format that can be consumed easily by other people.
  • Data repository: this helps researchers to browser datasets and/or contribute your own data easily. This central repository also enhances the discoverability of data and the associated research.
  • Tools: we also aim to develop tools that make it easier to standardize and share your data as well as tools for common data analysis.

Data Format Recommendations

Table for environmental data

  • High frequency time-series data
  • Logged automatically by sensors/actuators

Table for crop/biological data

  • Low frequency time-series data
  • Input by humans

Description file (metadata)

Necessary

  • Brief description of data, including date and location of the crop
  • Semantics and unit for each attribute
  • Data owner and contact info
  • Citation if published

Nice to have

  • Funding source
  • Specs of instruments and calibration method used
  • Known problems limiting the data use
  • Names of data sets related to this set
  • [anything available and considered useful to other researchers]
